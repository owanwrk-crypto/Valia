# 📖 Guía Rápida: Usar la Nueva Pestaña "Productos"

**¿Para qué sirve?** Registrar productos con **todos sus costos reales** para cotizaciones precisas.

---

## 🚀 Primeros Pasos

### 1. Ir a la Pestaña
```
Abrir aplicación → Hacer clic en "PRODUCTOS"
```

### 2. Ver Estadísticas
```
Arriba verás:
- Total de productos registrados
- Número de categorías
- Costo promedio de tus productos
```

---

## ➕ Agregar un Nuevo Producto

### Paso 1: Llenar Información Básica
```
Nombre: "Taza Personalizada Blanca"
Categoría: "Tazas" (elige del menú)
Descripción: "Taza blanca de 11oz con asa ergonómica"
```

### Paso 2: Registrar Costos Detallados
```
💲 Materia Prima: $15.00 (costo de la taza vacía)
👷 Mano de Obra: $8.00 (tiempo de impresión + personal)
🏭 Indirectos: $2.50 (electricidad, alquiler proporcional)
📊 Administrativos: $1.50 (contabilidad, marketing)
📦 Otros: $1.00 (embalaje, transporte)
```

### Paso 3: Ver Cálculo Automático
```
El sistema calcula automáticamente:
💰 COSTO TOTAL: $28.00 (suma de todos los costos)
```

### Paso 4: Definir Precio de Venta
```
🏷️ Precio Venta: $45.00 (lo que cobrarás al cliente)
Unidad: "pza" (pieza, docena, etc.)
```

### Paso 5: Guardar
```
Hacer clic en "GUARDAR PRODUCTO"
¡Listo! Aparecerá en la tabla abajo
```

---

## 📊 Ver y Gestionar Productos

### Buscar Productos
```
Escribir en "Buscar producto..." → Se filtra automáticamente
Ejemplo: "taza" mostrará todas las tazas
```

### Filtrar por Categoría
```
Clic en "Filtro: Todas" → Elegir categoría específica
Ejemplo: "Textiles" mostrará solo playeras, gorras, etc.
```

### Ver Márgenes
```
En la tabla verás:
- Costo Total: $28.00
- Precio Venta: $45.00
- Margen: 60.7% 🟢 (verde = ganancia)

Colores del margen:
🟢 Verde: Ganancia (>0%)
🔴 Rojo: Pérdida (<0%)
⚪ Blanco: Sin precio definido
```

### Editar Producto
```
Clic en el botón "Editar" del producto → Se cargan datos en formulario
Modificar lo necesario → "GUARDAR PRODUCTO"
```

### Eliminar Producto
```
Clic en "Eliminar" → Confirmar eliminación
⚠️ No se puede recuperar una vez eliminado
```

---

## 📊 Exportar Datos

### Exportar a Excel
```
Clic en "Export Excel" → Se descarga archivo automáticamente
Incluye todos los costos detallados + márgenes
Nombre del archivo: productos_YYYY-MM-DD.xlsx
```

## 💡 Consejos para Costos Reales

### Materia Prima
- Costo exacto del material base
- Incluye desperdicios si aplica
- Ejemplo: Taza = $12, Impresión = $3

### Mano de Obra
- Salario por hora ÷ productos por hora
- Incluye beneficios del empleado
- Ejemplo: $10/hora ÷ 10 productos/hora = $1 por producto

### Gastos Indirectos
- Electricidad, agua, internet
- Alquiler proporcional
- Mantenimiento de equipos

### Administrativos
- Contabilidad, marketing, teléfono
- Porcentaje del costo total (5-10%)
- Software y licencias

### Otros Costos
- Transporte y envío
- Embalaje y etiquetas
- Comisiones de venta

## ❓ Preguntas Frecuentes

### ¿Puedo cambiar los costos después?
**Sí**, edita el producto y guarda los cambios.

### ¿Qué pasa si dejo campos en blanco?
**Los costos en blanco se consideran $0.00**

### ¿Los productos aparecen en cotizaciones?
**Próximamente** - se integrará en la siguiente actualización.

### ¿Puedo tener productos inactivos?
**Sí**, pero por ahora todos son activos.

### ¿Cómo veo productos rentables?
**Ordena por margen** - los verdes son los más rentables.

## 🚨 Problemas Comunes

### "No se guardan los productos"
- Verificar conexión a internet
- Revisar que llenaste el nombre
- Contactar soporte si persiste

### "Los cálculos no salen bien"
- Revisar que no hay letras en campos numéricos
- Usar punto (.) para decimales, no coma (,)

### "No veo la pestaña Productos"
- Recargar la página (F5)
- Verificar que se actualizó el código
